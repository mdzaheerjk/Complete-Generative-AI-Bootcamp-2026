## 🚀 Getting Started

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/mdzaheerjk/Complete-Generative-AI-Bootcamp-2026.git
   cd Complete-Generative-AI-Bootcamp-2026
   ```

2. **Create a virtual environment**
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows: venv\Scripts\activate
   ```

3. **Install dependencies**
   ```bash
   pip install -r requirements.txt
   ```

4. **Start Jupyter Notebook**
   ```bash
   jupyter notebook
   ```

### Running Notebooks

- Open any `.ipynb` file in Jupyter
- Follow along with the tutorials
- Modify and experiment with code
- Execute cells sequentially
